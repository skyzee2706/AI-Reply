# Auto Yapping by Skyzee

Browser extension for Twitter/X that adds AI reply buttons (Agree, Ask, Funny, Debate) to tweet reply boxes.

## Features
- **Smart Reply Buttons**: Automatically generated replies inserted directly into Twitter's editor.
- **Multi-Provider Support**: Switch between **Groq** (Llama 3.1), **OpenAI** (GPT-4o), and **Gemini** (2.0 Flash), and many more.
- **Custom Persona**: Set your own system prompt to define how the AI should behave.
- **Privacy Focused**: API Keys are stored locally in your browser.

## Setup
1. Download this folder.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select this folder.

## Configuration
1. Click the extension icon in your browser toolbar.
2. Select your preferred AI Provider.
3. Enter your API Key.
4. (Optional) Customize the "Base Persona".
5. Click **Save Settings**.

## Usage
1. Open any Tweet on X.com.
2. Click the **Reply** icon.
